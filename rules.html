Sudoku

Rules

  • Fill the numbers from 1 to 9 in each row
  • Put in each column the numbers from 1 to 9
  • Fill in every 3 x 3 box the numbers from 1 to 9

    Hints: Each completely filled row must contain the numbers from 1 to 9.

    Only one number may be entered in each cell.

    It is possible to enter several possible candidates in a cell.

    A solved field contains only a number between 1 and 9.

    The default digits can not be changed.

    The sudoku solution is to build around these numbers.
